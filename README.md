# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/pedron3to/react-native-example.git
```

```bash
$ cd react-native-example
# Install Dependencies
$ yarn 
```

```bash
$ yarn start
```

Go to http://localhost:19006 to see the result.
